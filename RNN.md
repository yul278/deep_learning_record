# RNN
1. 能使用不同数量的时间序列实现预测
2. 跟普通的网络相比加上了反馈循环，从激活函数重新加偏置加到激活函数的重新输入中
3. 但是过多运算将会导致梯度消失/爆炸：靠前面层的数据一直重复乘权重加往下层，权重一直加乘

# LSTM 长短期记忆网络
1. 不使用相同的反馈环链接，除了使用RNN的短期记忆的连接外，再加入一条直接连到输出的长期记忆链接
2. 使用sigmod/Tanh激活函数
3. 每次运算分为两部分，一部分决定输出数值，另一部分决定加入长期记忆的权重，两者相乘加入长期记忆
4. 决定原始长期记忆百分比，决定记住潜在长期记忆百分比，短期记忆百分比，用sigmod
- 单元运算步骤
    1. 遗忘门：接收输入和短期记忆，运算出一个权重决定从上一个单元继承多少长期记忆
    2. 输入门：同样接受输入和短期记忆，一部分决定存入新信息存入权重，另一部分决定新记忆内容
    3. 更新长期记忆：将前两部分内容分别乘完权重相加，新的长期记忆、
    4. 输出门：再次使用短期记忆和输入经运算和sigmod得权重，与新长期记忆过Tanh得内容相乘得输出，同时作为新短期记忆传出
