# CNN
convolutional neural network 卷积神经网络

1. 三件事实现图像分类
   1. 减少输入节点数量
   2. 可以容忍图像中像素位置的微小变化
   3. 观察复杂图像发现的相关性

2. 工作过程
   1. 通常首先使用3*3的像素过滤器/卷积核 其中每个像素的强度由反向传播确定   
   2. **提取特征**：每次计算输入和卷积核的卷积结果，输出加上偏置，最后的值转化为特征图
   3. 持续移动完成特征图，每次可以移动一个像素位置或更多
   4. 特征图经过激活函数后得到新的特征图
   5. **池化**：可以对每一部分（不重叠）取最大值（最大池化），会选择卷积核发挥最佳作用的位置，即取到最大值的那一次卷积位置
        - 均值/平均池化
   6. 加入正常的神经网络进行权重偏置激活输出
